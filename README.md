## <img src="assets/features.svg" width="26" align="top" /> Features

Noteum is designed for developers, students who need a beautiful and fast place to organize their lives.

- <img src="assets/markdown.svg" width="20" align="top" /> **Advanced Markdown Editor**: Full GitHub flavored Markdown support with math (KaTeX), syntax highlighting, and tables.
- <img src="assets/slash.svg" width="20" align="top" /> **Slash Commands**: Type `/` to instantly insert lists, code blocks, math blocks, and formatting without losing flow.
- <img src="assets/calendar.svg" width="20" align="top" /> **Integrated Calendar**: A beautiful grid calendar seamlessly connected to your daily notes.
- <img src="assets/ai.svg" width="20" align="top" /> **AI Assistant**: Built in AI chat using multiple powerful models (Gemini, Llama) to help you write and code.
- <img src="assets/ui.svg" width="20" align="top" /> **Modern UI**: Pure simplicity. Switch between light and dark mode instantly

## <img src="assets/architecture.svg" width="26" align="top" /> Architecture

Noteum is built with a highly performant stack:
- **Backend**: Rust (Axum) ensuring maximum memory safety and performance.
- **Frontend**: Vanilla JS + HTML5 + CSS3 (Zero heavy frameworks).
- **Desktop**: Tauri (Lightweight and native WebVew).
